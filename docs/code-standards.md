# Code Standards

**Structure:** Feature-based (src/components, src/pages, src/services).

**React:** Hooks only (useState/useMemo/useCallback), functional components, no class.

**JS/TS:** ES6+, camelCase, const/let, no var.

**CSS:** Custom (variables/Grid/Flex), mobile-first, a11y contrast.

**Perf:** Memo expensive ops, no re-renders.

**A11y:** ARIA, semantic HTML, keyboard nav.

**Git:** Descriptive commits, no force-push.

**Tests:** Vitest 80%+ coverage.

**Deps:** Minimal (React, Router, Gemini lib backend).
