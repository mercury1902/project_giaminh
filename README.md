# Lịch sử Việt Nam — React Timeline

Ứng dụng React giao diện hiện đại (tông màu sáng) với timeline sự kiện lịch sử Việt Nam, bộ lọc, zoom, điều hướng bàn phím và tìm kiếm.

## Chạy dự án

Yêu cầu: Node 18+

```bash
npm install
npm run dev
```

Build sản phẩm:

```bash
npm run build
npm run preview
```

## Tính năng
- Trang chủ, Khái quát, Timeline tương tác, Tìm kiếm
- Chủ đề sáng, tương phản tốt, tối ưu trải nghiệm
- Timeline: lọc theo thời kỳ/triều đại, zoom, điều hướng trái/phải
- Tìm kiếm: theo tiêu đề, mô tả, triều đại, năm, thời kỳ
- Hỗ trợ ARIA, điều hướng bàn phím, responsive

## Cấu trúc thư mục
- `src/App.jsx`: giao diện chính, các section
- `src/data/events.js`: dữ liệu sự kiện, thời kỳ, triều đại
- `src/styles.css`: phong cách hiện đại, sáng
